¿Qué es una Función Flecha?

Una función flecha es simplemente una forma más corta y moderna de escribir una función en JavaScript. Hace lo mismo que una función normal, pero con menos palabras.

¿Por qué se usan?

  • Son más rápidas de escribir.
  • El código se ve más limpio.
  • Funcionan muy bien cuando queremos usar funciones pequeñas.

¿Qué es una Promesa?

Una promesa es algo que JavaScript usa para trabajar con tareas que tardan un tiempo en completarse, por ejemplo: cargar datos, esperar una respuesta del servidor o leer un archivo.

Una promesa significa: "Todavía no tengo el resultado, pero te lo entregaré más adelante".

Estados de una promesa

  • Pendiente: JavaScript sigue trabajando.
  • Resuelta: Todo salió bien.
  • Rechazada: Algo falló.
